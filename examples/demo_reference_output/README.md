# Demo Reference Output

This directory contains sample output files from running the comprehensive demo for reference purposes.

## Files

### demo_report.html
The HTML report generated by the demo. Open this file in a web browser to see the full interactive report with:
- Summary statistics
- Detailed test results for each component
- Visual diff metrics
- Remediation suggestions

To view:
```bash
# On macOS
open demo_report.html

# On Linux
xdg-open demo_report.html

# On Windows
start demo_report.html
```

### demo_results.json
The structured JSON output containing all test findings. This can be used for programmatic analysis or integration with other tools.

Example:
```bash
cat demo_results.json | python -m json.tool
```

### dashboard_baseline_sample.txt
A sample baseline image file showing the text format used to store visual baselines. Each line represents a row of pixels, with space-separated grayscale values (0-255).

### dashboard_diff_sample.txt
A diff matrix generated for the dashboard component when significant changes are detected. Higher values indicate greater pixel differences.

## Regenerating Outputs

To generate fresh outputs from the demo, run:

```bash
python examples/comprehensive_demo.py
```

The new outputs will be placed in:
- `examples/demo_sample_output/` - HTML and JSON reports
- `examples/demo_visual_artifacts/` - Baseline and diff images

Note: These directories are git-ignored and will not be committed to the repository.
